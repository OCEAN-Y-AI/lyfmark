# Modulbeispiele (Kundendoku)

## Zweck

Dieser Ordner enthält direkt nutzbare Beispiel-Snippets für Module.

## Nutzung

- Modul auswählen: `docs/public/modules/overview.md`
- Passendes Snippet hier öffnen und in die Seite übernehmen.
- Pro Modul gibt es in der Regel eine gleichnamige Beispieldatei (z. B. `split.md`).
