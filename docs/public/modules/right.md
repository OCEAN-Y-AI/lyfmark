# Modul `right`

## Zweck

Richtet einen Block am rechten Seitenrand aus, ohne den Text selbst rechtsbündig zu setzen.

## Wann einsetzen

Nutzen, wenn ein kurzer Hinweis oder CTA am Rand stehen soll (z. B. Kontaktbox, Hinweis).

## Verwendung (Minimal‑Beispiel)

```md
:::right

### Direkter Kontakt

[Jetzt anrufen](tel:+49301234567)

:::
```

## Optionen

- `label` – kurze Beschreibung des Blocks (nützlich bei mehreren rechten Boxen).
- `style` – für Sonderfälle (individuelles CSS).

## Stolperstellen

- Auf sehr kleinen Bildschirmen richtet sich der Block automatisch links aus.
